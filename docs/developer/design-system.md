# Warm Instrument design system

genie's locked visual language is **Warm Instrument**: warm bone paper, near-black ink,
hairline structure, and a restrained clay accent.

The identity rule is simple: clay appears only on generation and refine moments.
Structural navigation, panels, save actions, and ordinary primary buttons stay ink or
neutral. Embedded cards use system fonts under the strict MCP-App CSP; standalone
surfaces may use the brand type stack.

The implementation tokens live in `docs/designs/design-6/tokens.css`. Current SVG
references in that directory document the front door, UI-kit browser, refine surface,
embedded modes, primitives, foundations, and genie card.
